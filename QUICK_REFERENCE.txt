🎯 COMMON QUESTIONS
═══════════════════════════════════════════════════════════════════════════════

Q: Can I use it without OpenAI API?
A: YES! It works perfectly without it. Local responses are still smart!

Q: Why is first startup slow?
A: Normal - AI models are being downloaded/loaded. ~1 minute first time.

Q: What if I get "ModuleNotFoundError"?
A: Run: pip install -r requirements.txt

Q: How do I update college data?
A: Add more notices to output/notice_dataset.json, then restart.

Q: Is my data secure?
A: Data stays on your computer. Only queries go to OpenAI (if enabled).

Q: Can multiple people use it?
A: Yes! Run on local network: python app.py

Q: Does it work offline?
A: Yes! Without OpenAI API key it's 100% offline.
