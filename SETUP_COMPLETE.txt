🔧 OPTIONAL: ENABLE CHATGPT FOR BETTER RESPONSES
═══════════════════════════════════════════════════════════════════════════════════

1. Get API key from: https://platform.openai.com/api/keys

2. Set environment variable:
   
   Windows (PowerShell):
   $env:OPENAI_API_KEY = "sk-your-key-here"
   
   Windows (Command Prompt):
   set OPENAI_API_KEY=sk-your-key-here
   
   Mac/Linux:
   export OPENAI_API_KEY=sk-your-key-here

3. Start the server:
   python app.py

4. Now you get ChatGPT-powered responses!

(Without the key, the system still works great with local smart responses)
